
## Instalación

Para instalar el proyecto, lo descargamos y luego ejecutamos

```bash
  npm install
```
    
## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
